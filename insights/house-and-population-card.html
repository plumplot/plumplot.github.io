House, Population Card

If you select any map under houses ▾ menu and click on some postcode sector a house card pops up.

  1. Filter setting is displayed under the postcode sector name.
  2. There are additional tabs with reports.
  3. On the first tab (Area comparison), there is a link to rightmove.co.uk which shows you sold properties in a given sector based on your filter settings.

A population card pops up if you select any map from population ▾ menu.
